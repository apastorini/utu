## 🤖 Patrón de Agentes

Este proyecto utiliza un patrón de agentes con 5 archivos clave:

| Archivo | Función | Actualización |
|---------|---------|---------------|
| **INDICE.md** | Navegación y patrón de agentes | Manual |
| **MEMORY_BANK.md** | Contexto persistente del proyecto | Manual |
| **AGENT_RULES.md** | Reglas para crear clases | Manual |
| **PROJECT_STATE.json** | Estado actual del proyecto | Automática |
| **STATUS.md** | Progreso, changelog y tareas | Manual |

### Flujo de Trabajo
1. Agente lee **MEMORY_BANK.md** para contexto
2. Agente consulta **AGENT_RULES.md** para estructura
3. Agente crea clase siguiendo plantilla
4. Agente actualiza **PROJECT_STATE.json** y **STATUS.md**
5. Archivo completado lleva sufijo de iteración: `COMPLETADO_v1.md`
