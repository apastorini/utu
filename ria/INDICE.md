## 🎯 Características del Curso

### Seguridad desde el Inicio
- ✅ HTTPS y CORS configurados
- ✅ Validación y sanitización de inputs
- ✅ JWT con refresh tokens
- ✅ Rate limiting
- ✅ OWASP Top 10

### Usabilidad y Accesibilidad (AGESIC)
- ✅ WCAG 2.1 AA compliance
- ✅ Navegación por teclado
- ✅ Lectores de pantalla
- ✅ Contraste de colores

### Localización (i18n)
- ✅ Soporte multiidioma
- ✅ Formatos de fecha/hora
- ✅ Monedas locales

### Mobile First
- ✅ Responsive design
- ✅ Touch-friendly
- ✅ Performance optimizado

### Proyecto Integrado
- **Sistema de Eventos** (similar a tufiesta.uy)
  - Crear eventos
  - Buscar y filtrar
  - Comprar entradas
  - Gestión de usuarios
  - Panel de administrador

## 🛠️ Stack Tecnológico

### Frontend
- HTML5, CSS3, JavaScript ES6+
- React 18 con Hooks
- Bootstrap 5 + Material-UI
- i18next para localización

### Backend
- Node.js 18+, Express.js
- MongoDB + Mongoose
- JWT, gRPC, Protocol Buffers
- Promesas, Async/Await

### DevOps
- Docker, Docker Compose
- Nginx, Environment variables
