## 🛠️ Stack Tecnológico

### Frontend
- **HTML5** - Semántica y accesibilidad
- **CSS3** - Flexbox, Grid, Animations
- **JavaScript ES6+** - Moderno y funcional
- **React 18** - Hooks, Context, Suspense
- **Bootstrap 5** - Componentes responsive
- **Material-UI** - Componentes avanzados
- **i18next** - Localización

### Backend
- **Node.js 18+** - Runtime
- **Express.js** - Framework web
- **MongoDB** - Base de datos NoSQL
- **Mongoose** - ODM
- **JWT** - Autenticación
- **Joi** - Validación

### DevOps
- **Docker** - Containerización
- **Docker Compose** - Orquestación local
- **Nginx** - Proxy reverso
- **Environment variables** - Configuración
